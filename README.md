**VAWCare** is a **Progressive Web Application (PWA)** developed to support **barangay-level** efforts in preventing and responding to **Violence Against Women and Children (VAWC)** cases.  
It provides a **confidential**, **accessible**, and **community-driven** digital platform where victims, citizens, and officials can engage, report, and act efficiently.

* **Emergency Alert System** – Instantly notify barangay officials during emergencies.  
  This feature ensures that victims can quickly alert authorities, enabling rapid response and coordination within the barangay.

* **Chatbot Assistance** – Get confidential support and guidance on VAWC-related issues.  
  The chatbot provides step-by-step guidance, answers common questions, and connects users to the right resources without compromising privacy.

* **Resource Hub** – Access educational materials, rights information, and reporting steps.  
  Offers curated articles, legal guidance, and preventive tips to empower citizens and victims with knowledge about VAWC and their rights.

* **Decision Support System (DSS)** – View analytics and insights on reported cases.  
  Helps barangay officials monitor trends, identify high-risk areas, and make data-driven decisions for interventions.

## Technologies Used

### Frontend

**Language:** JavaScript / TypeScript  
**Framework:** React – Core library for building user interfaces.  

**UI:**  
- Ant Design (antd) – UI component library and design system.  
- Recharts – Charts and data visualization.  
- Framer Motion – Animations and smooth UI transitions.
- React Bits - UI compenent Library

**Routing & Networking:**  
- React Router DOM – Client-side routing and navigation.  
- Axios – HTTP client for API requests.  

**Maps:**  
- Leaflet + React Leaflet – Interactive maps for geolocation features.  
- React-Leaflet-MarkerCluster – Clustered markers for map efficiency.  

**Backend Integration:**  
- Firebase – Authentication, Firestore, and backend integration.  

### Backend

**Language / Runtime:** Node.js – Server-side JavaScript runtime  
**Framework:** Express – REST API framework for building server endpoints.  

**Database & Data Management:**  
- Mongoose – MongoDB object modeling and database integration.  
- JSON Rules Engine – Rule-based logic for Decision Support System (DSS).  

**Authentication & Security:**  
- Firebase Admin SDK – Server-side Firebase operations.  
- Bcryptjs – Password hashing and security.  

**Utilities:**  
- Dotenv – Environment variable management.  
- Cors – Enable cross-origin requests.  
- Nodemailer – Sending emails and alerts from the backend.
